### 3조: 야구 중계플레이 Wiki 앱
### 최승표, 문중협, 김영

### To start programming
Run the following commands in Ubuntu to download the skeleton code for the project.
```shell
git clone https://github.com/vietman2/2023W-Group2
```

### To Start Development Server
```shell
cd app
yarn install
yarn start
```
